# ChildGuard AI — ID Protector Card Generator (React SPA)

Modern React architecture application for generating, customizing, downloading, and printing digital child and parent safety ID cards. Built with React 18, Vite, Tailwind CSS, Lucide icons, html2canvas, and QRCode.js.

---

## 🚀 Quick Start

### 1. Install Dependencies
```bash
npm install
```

### 2. Run Locally in Development
```bash
npm run dev
```

### 3. Build for Production
```bash
npm run build
```

---

## 🌐 Deploying to Render / Vercel / Netlify

### Option A: Render (Static Site)
1. Go to [Render Dashboard](https://dashboard.render.com/).
2. Click **New +** -> **Static Site**.
3. Select GitHub repository: `https://github.com/harshithharshith8513-design/child_monitoring_react`.
4. Configure site settings:
   - **Name**: `child-monitoring-react`
   - **Build Command**: `npm run build`
   - **Publish Directory**: `dist`
5. Click **Create Static Site**.

### Option B: Vercel
1. Go to [Vercel Dashboard](https://vercel.com/new).
2. Import repo `child_monitoring_react`.
3. Vercel auto-detects Vite:
   - **Framework Preset**: Vite
   - **Build Command**: `npm run build`
   - **Output Directory**: `dist`
4. Click **Deploy**.

### Option C: Netlify
1. Go to [Netlify App](https://app.netlify.com/).
2. Import from GitHub `child_monitoring_react`.
3. Build Settings:
   - **Build command**: `npm run build`
   - **Publish directory**: `dist`
4. Click **Deploy Site**.
