ChildGuard AI Blocker

Install the ChildGuard AI Blocker extension to block blacklisted domains across all websites. This extension runs locally, automatically synchronizes with your Dashboard blacklist, and integrates **face-scan unlocking** using the Human AI biometric engine so parents can verify their identity and bypass blocks in new browser tabs instantly!

How to install & use:

  1. Ensure you have registered a Parent biometric profile under the Facial Mapping section.
  2. Download the Extension ZIP file using the button on the right and extract the folder.
  3. Open Google Chrome (or Edge, Brave) and navigate to chrome://extensions/.
  4. Toggle Developer mode on in the top-right corner of the page.
  5. Click Load unpacked in the top-left and select the extracted folder directory.
  6. Open the extension toolbar popup and click Sync from Server to download your blacklist and biometric keys.
  7. Visit any blocked website in a new tab. It will redirect to a custom block screen. Have the parent click Scan & Unlock to verify their face and bypass the block for 15 minutes!
